## 📌 Projects Included

### 1️⃣ AI Resume Screening System
A semantic resume analyzer that intelligently matches resumes with job descriptions.

#### 🔍 Overview
Traditional systems rely on keyword matching, often missing qualified candidates. This system uses NLP to understand context and meaning, ensuring better candidate-job alignment.

#### ⚙️ Tech Stack
- Python
- Streamlit
- Hugging Face (Sentence-BERT: all-MiniLM-L6-v2)
- PyPDF2
- Scikit-learn (Cosine Similarity)
- NumPy

#### ✨ Key Features
- Semantic matching instead of keyword filtering
- Batch resume processing
- Match scoring with ranking system:
  - ✅ Strong Match (80%+)
  - ⚠️ Potential Match (50–79%)
  - ❌ Low Match (<50%)
- Interactive UI

### 2️⃣ AI Code Reviewer
An intelligent system that reviews code and provides feedback on quality, readability, and potential improvements.

#### 🔍 Overview
This tool combines rule-based analysis with AI-based suggestions to help developers improve code quality and follow best practices.

#### ⚙️ Tech Stack
- Python
- Streamlit
- Transformers (Hugging Face)
- Sentence Transformers
- Static Code Heuristics

#### ✨ Key Features
- Code quality scoring
- Readability analysis
- AI-based improvement suggestions
- Supports multiple programming languages
- Real-time feedback interface
